Data types




Understanding

To demonstrate understanding of data types, create a simple C program (You can use Visual studio) with each data type listed in the video and debug the program with Ollydbg to see how the values are stored in assembly. Here is an excellent Ollydbg tutorial.

Practice

For practice, try declaring multiple variables of the same type and in different orders and practice finding the entry point to the main function in these various scenarios. Then observe any differences in how each data type is declared in assembly.

Notes

In Visual studio 2015, you can enable compiler optimizations and disable exploit mitigations through the [Project]->[YourProjectName Properties] menu. Enable the compiler optimizations [C/C++->Optimizations] Optimization: /O1 (minimize for size), Security Check: Disable Security Check /GS- ; [Linker->General] Enable Incremental Linking: No (/INCREMENTAL:NO).

Binary and Hexadecimal Digits




Understanding

Read THIS article. To demonstrate understanding of binary and hexadecimal digits, explain the bases of the different number systems to a friend. What is the base of each number system and how does it account for decimal having the digits 0-9, binary having the digits 0-1, and hexadecimal having the digits 0-F?

Practice

For practice, write down the first 10 numbers, between 0 and 15, that come to mind in decimal, then convert them first to binary, and then to decimal. Try all combinations of this exercise: decimal->binary->hexadecimal, decimal->hexadecimal->binary, hexadecimal->decimal->binary, hexadecimal->binary->decimal, binary->hexadecimal->decimal, binary->decimal->hexadecimal.
